## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+ and npm

### Installation

```bash
# Clone the repository
git clone https://github.com/gex82/minecraft_old_SJU.git
cd minecraft_old_SJU

# Install dependencies
npm install

# Start development server
npm run dev
```

Open your browser to `http://localhost:5173`

### Controls
- **W/A/S/D** - Move
- **Mouse** - Look around
- **Space** - Jump
- **Esc** - Pause menu
